Usage Notes

1) Week 2 Verb Priority

Use regular -ar verbs to describe classroom habits and routines before adding tense complexity. Core productive forms: yo, tÃº, Ã©l/ella/usted, nosotros.

2) Keep Week 1 Identity Active

Start many Week 2 interactions with Week 1 identity frames (soy..., soy de...) and then continue with Week 2 action language (estudio..., practico...).

3) Negation Placement

Place no directly before the conjugated verb: No estudio por la noche. Avoid English order transfer.

4) Formal vs Informal Classroom Address

Use usted forms when speaking politely to the teacher in formal practice tasks; use tÃº with classmates in peer interactions.

Mini Practice Prompts (Vocabulary Retrieval)

Prompt Set A - Oral Speed Rounds

  1. Say four actions you do in class using yo forms.
  2. Ask your partner three yes/no study routine questions.
  3. Answer each question with one extra detail (time or place).
  4. Make two negative routine sentences.
  5. End with a polite closing expression.

Prompt Set B - Written Retrieval

  1. Write six sentences using verbs from Category 1.
  2. Write four sentences using nouns from Category 2.
  3. Write four routine lines using Category 3 time phrases.
  4. Write a short teacher-student clarification exchange using Category 4 expressions.

Prompt Set C - Integration with Week 1

  • Introduce yourself (name, origin, role) and then describe your class routine.
  • Ask where your partner is from and when they study.
  • Present your partner to the class using one ser sentence and two -ar action sentences.
