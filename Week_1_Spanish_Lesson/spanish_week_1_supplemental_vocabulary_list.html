Usage Notes (Week 1 Focus)

2) Profession Article Rule for Beginner Production

Mini Practice Prompts (Supplemental Vocabulary Retrieval)

Prompt Set A - Quick Oral Retrieval

  1. Say three greetings and one closing expression.
  2. Introduce yourself with name, origin, and role.
  3. Ask two different identity questions.
  4. Say one sentence with tambi√©n and one with pero.
  5. Introduce one friend to the class.

Prompt Set B - Short Writing Retrieval

  1. Write five lines using vocabulary from Category 1.
  2. Write four lines using vocabulary from Category 2.
  3. Write four lines that include one country and one nationality each.
  4. Write a 6-line dialogue using at least three question words/connectors.

Prompt Set C - Partner Interaction

  • Partner A asks: name, origin, and role.
  • Partner B answers and asks follow-up questions.
  • Switch roles and repeat with new vocabulary.
  • Each partner presents the other in third person using es.
