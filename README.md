# Spanish Lessons - 12-Week Beginner Curriculum

A complete beginner Spanish program with weekly lesson packages for classroom teaching and self-study.

## Live Access

- **GitHub Pages (primary):** https://weida8.github.io/Spanish-Lessons/index.html
- Start from the landing page to browse all 12 weeks and open materials in sequence.

## Project Overview

This repository provides a structured 12-week Spanish curriculum (beginner level, A1 toward A1+) with practical learning resources for each week.

Each weekly package includes:

- **Lesson Plan** - Instructor-ready daily flow and teaching sequence.
- **Workbook** - Guided and independent practice tasks.
- **Answer Sheet** - Model answers and checking support.
- **Supplemental Vocabulary List** - Theme vocabulary and reinforcement prompts.

## Who This Repository Is For

### Learners

- Study week by week from Week 1 to Week 12.
- Follow the same within-week order for consistency:
  1. Lesson Plan
  2. Workbook
  3. Answer Sheet
  4. Supplemental Vocabulary List

### Teachers

- Use the Lesson Plan as your core instruction guide.
- Assign Workbook activities for classwork/homework.
- Use Answer Sheets for feedback and quick checks.
- Use Supplemental Vocabulary Lists for extension and review.

## Local Access (Fallback)

1. Clone or download this repository.

## Repository Layout

```text
.
├── index.html
├── styles/
│   └── weekly-shared.css
├── Week_1_Spanish_Lesson/
├── Week_2_Spanish_Lesson/
├── ...
└── Week_12_Spanish_Lesson/
```

Within each `Week_<N>_Spanish_Lesson/` directory, files follow this naming pattern:

- `spanish_week_<n>_lesson_plan.html`
- `spanish_week_<n>_workbook.html`
- `spanish_week_<n>_answer_sheet.html`
- `spanish_week_<n>_supplemental_vocabulary_list.html`

## Contributor Notes (Short)

- Keep week folder and file naming conventions consistent.
- Update `index.html` whenever files are added, renamed, or reorganized.
- Keep weekly resources aligned as a four-document package.
